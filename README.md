Dieser Bot dient mir, von unterwegs schnell mal ein Host zu Pingen oder eine DNS anfrage zu senden.
Er wird bei gelegenheit um weitere Optionen erweitert.

## Aktuelle Befehle

/help - Help
/echo ... - Sendet ... zurueck
/ping ... - Pingt .... an
/ns domain [type] [dns] - DNS Abfrage type=a, dns=8.8.8.8
/fortune - Fortune!
